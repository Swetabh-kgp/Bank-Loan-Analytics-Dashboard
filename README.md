# Bank Loan Analytics Dashboard | Power BI
## Project Overview
This project is an interactive Bank Loan Analytics Dashboard developed using Microsoft Power BI.
The dashboard provides insights into loan applications, funded amounts, repayments, interest rates, debt-to-income ratios, loan purposes, home ownership status, and geographical distribution of loan applications.
The objective of this project is to transform raw banking data into actionable business insights that support decision-making and loan performance monitoring.

## Key Business Insights -
- 86.18% of loan applications are categorized as Good Loans.
- 13.82% of applications fall under Bad Loans.
- Total Funded Amount reached $436M.
- Total Amount Received reached $473M.
- Debt Consolidation is the most common loan purpose.
- Most borrowers prefer 36-month loan terms.
- Loan applications show steady growth throughout the year.
